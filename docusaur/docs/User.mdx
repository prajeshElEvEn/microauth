<p>Mongoose User Schema representing user details.</p>

**Type**: Object

|PROPERTY|TYPE|DESCRIPTION|
|:---:|:---:|:---:|
|firstName|string|<p>The first name of the user.</p>|
|lastName|string|<p>The last name of the user.</p>|
|email|string|<p>The email address of the user (unique).</p>|
|role|"admin" \| "user"|<p>The role of the user, either &quot;admin&quot; or &quot;user&quot; (default: &quot;user&quot;).</p>|
|password|string|<p>The password of the user.</p>|
|avatar|string|<p>The URL or path to the user's avatar image.</p>|
|resetPasswordToken|string|<p>Token for resetting the user's password.</p>|
|resetPasswordExpires|Date|<p>Expiration date for the reset password token.</p>|
|createdAt|Date|<p>Timestamp representing the creation date.</p>|
|updatedAt|Date|<p>Timestamp representing the last update date.</p>|
