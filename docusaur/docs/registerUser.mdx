```ts
function registerUser(req: object, res: object)
```

<div>
  <span className="badge badge--success">async</span>
</div>

<p>This function handles the registration of a new user.</p>

**Returns**: void

|PARAMETER|TYPE|OPTIONAL|DEFAULT|DESCRIPTION|
|:---:|:---:|:---:|:---:|:---:|
|req|object|||<p>Express request object.</p>|
|res|object|||<p>Express response object.</p>|
|req.body.firstName|string|||<p>First name of the user.</p>|
|req.body.lastName|string|||<p>Last name of the user.</p>|
|req.body.email|string|||<p>Email address of the user.</p>|
|req.body.password|string|||<p>User's password.</p>|

**Throws**: Error

<p>Will throw an error if required fields are not provided,<br />if the user already exists, or if registration fails.</p>

```app.post('/api/v1/auth/register', registerUser);```
