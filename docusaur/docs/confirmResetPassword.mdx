```ts
function confirmResetPassword(req: object, res: object)
```

<div>
  <span className="badge badge--success">async</span>
</div>

<p>This function confirms the reset of a user's password.</p>

**Returns**: void

|PARAMETER|TYPE|OPTIONAL|DEFAULT|DESCRIPTION|
|:---:|:---:|:---:|:---:|:---:|
|req|object|||<p>Express request object.</p>|
|res|object|||<p>Express response object.</p>|
|req.params.token|string|||<p>Token from the reset email.</p>|
|req.body.newPassword|string|||<p>New password for the user.</p>|

**Throws**: Error

<p>Will throw an error if required fields are not provided,<br />if the token is invalid or expired, or if password reset fails.</p>

```app.post('/api/v1/auth/reset/:token', confirmResetPassword);```
